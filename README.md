# 📝 Notes Manager App

A modern and aesthetic CRUD notes application with **dark mode toggle** and **glassmorphism UI**.

## ✨ Features

* ➕ Add notes
* 👀 View notes
* ✏️ Edit notes
* 🗑 Delete notes
* 🌙 Dark / Light mode toggle
* 🧊 Glassmorphism design
* 💾 Data stored using localStorage

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
